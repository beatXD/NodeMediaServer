**How To run On local:**

1. Install dependencies

   ```bash
   npm install
   ```

2. Run development

   ```bash
   npm run dev
   ```

**How To run On Docker:**

1. Docker-compose up

   ```bash
   npm run docker:up
   -- or --
   docker-compose up -d --build
   ```

2. Docker-compose down

   ```bash
   npm run docker:down
   -- or --
   docker-compose down
   ```

**Server endpoint:** _`http//:localhost:8000`_
**Streaming endpoint:** _`http//:localhost:1935`_
